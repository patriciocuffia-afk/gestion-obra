⚙ Configuración de IA

La función de lectura de remitos usa la API de Claude (Anthropic). Ingresá tu clave API para habilitarla. Tu clave se guarda solo en este dispositivo.

Podés obtener tu clave en console.anthropic.com → API Keys. El uso de la API tiene un costo mínimo por imagen analizada (fracción de centavo).
